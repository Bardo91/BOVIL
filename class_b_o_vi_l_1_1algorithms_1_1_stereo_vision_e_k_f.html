Detailed Description

Implementation of Extended Kalman Filter that fuse stereo camera information to estimate position of an object.

In this EKF the observed state is compoud by four integers that are the pixel's X & Y of the targeting object in the images. Make sure that the image has the origin (0,0) at the top on the left. If not, a transformation is necessary. Another assumtion is that the Z coordinate of the camera is upfront and the X edge has the same orientation of the X edge of the image (X_c == X_img).
